# Pokemon Vocabulary

Authoritative glossary for nicknames and mechanic shorthand used when
discussing Pokemon. Prefer these expansions whenever a term appears in
user input, logs, or agent reasoning — do not leave slang unresolved.

---

## How to use

1. When a term from this glossary appears, expand it to the canonical
   meaning before reasoning or responding.
2. When defining a term for the user, use the definition column verbatim
   (you may add brief context after).
3. If a term is ambiguous and not listed here, say so — do not invent
   slang expansions.
4. To extend this skill, add a row to the matching table below (keep
   term → definition columns only).
